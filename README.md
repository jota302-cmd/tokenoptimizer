# SlimContext 📉

**SlimContext** (TokenOptimizer) es una utilidad diseñada para optimizar prompts de LLMs, reduciendo redundancias y stop-words para un uso más eficiente y económico de tokens.

## 🚀 Características

- **Token Slashing**: Elimina palabras innecesarias y redundancias del contexto sin perder el significado central.
- **Ahorro de Costos**: Ideal para enviar megacontextos a APIs de OpenAI o Anthropic reduciendo la facturación.
- **Zero-Server Processing**: Todo el procesamiento algorítmico ocurre en tu dispositivo local.

## 🛠️ Especificaciones Técnicas

- **Stack**: HTML5, Tailwind CSS, Vanilla JavaScript.
- **Diseño**: Minimalista y funcional.
- **Despliegue**: Subdominio Vercel (`optimizer.optimicrotools.com`).
